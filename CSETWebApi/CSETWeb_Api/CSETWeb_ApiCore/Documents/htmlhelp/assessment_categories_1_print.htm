Assessment Categories

 
 
The Assessment Configuration screen includes the Assessment Categories section where users select from a list of available assessment categories. The assessment category description can be expanded with the drop-down arrow to the right of each category. The following assessment categories are available:
 
Cybersecurity Assessment Module: A CSET Cybersecurity Assessment Module is based on:
 
  • Organizational Maturity (Maturity Model)
    A maturity model is a formal measurement used by an organization to gauge and improve its programs and processes. Maturity models are intended to measure the degree to which an organization has institutionalized its cybersecurity practices. Implementing process maturity within an organization will ensure that practices are consistent, repeatable, and constantly being improved.
     
  • Best Practices and Critical Infrastructure / Industry
Custom assessments similar in style and presentation to a CSET Maturity Model, but focused less on operational maturity and more on industry best practices and critical infrastructure resilience. These types of assessments include the CISA Ransomware Readiness Assessment, CISA Critical Infrastructure Survey, etc.
 
Standard-Based Assessment
A CSET standard-based assessment is based on industry standards like the NIST SP 800 series, the CSF, NERC, NISTIR and other industry authorities. The assessment examines the organization's cybersecurity posture against the standard, tests its security controls, and measures how they stack up against known vulnerabilities. Multiple standards can be selected to create one CSET assessment. 
 
Network Diagram
A network diagram is a visual representation of a computer or network. It shows the components and how they interact, including routers, devices, hubs, firewalls, etc. and can help define the scope of the network for the assessment.
 
Select the type of assessment category you wish to select your assessment from, and click next.
