AWWA and CSET: Joining forces for a comprehensive cybersecurity assessment package
 
The AWWA Cybersecurity Tool provides a spreadsheet-based assessment that is easy to use and familiar to the AWWA community. In order to build upon that valuable data to offer a better user experience and a more comprehensive cybersecurity analysis, AWWA and CSET combined efforts to create the new AWWA-specific CSET standard.
 
 
Before this joint effort, if an AWWA member wanted to conduct an assessment using both the AWWA tool and CSET, a significant amount of redundant work and duplicate effort would be necessary. 
 
To help streamline that process, the CSET team integrated the disparate applications to create one fully optimized AWWA/CSET package. The completed AWWA tool along with all the work performed in the original assessment can now be uploaded directly into CSET—with no duplicate effort or rework required. This integration brings the AWWA Cybersecurity Tool into a more comprehensive, sophisticated standard offering network diagram analysis and cybersecurity maturity model assessment in addition to regular interface maintenance, updates, references and source material, and a helpful user guide.
 
As an AWWA tool user you can import your AWWA results into CSET to determine the additional functionality you desire, such as network diagramming, additional standards, or assessment aggregation—and all of that comes together in a single assessment without any rework. On the other hand, if working with CSET initially seems overwhelming and the AWWA tool is more familiar to the user, the process of integrating data with CSET is both user-friendly and intuitive from the start. 
 
To interact effectively with AWWA and CSET, follow these guidelines:
 
 
Prepare Step:
 
Complete your organization’s Assessment Configuration details in the Assessment Configuration screen.
 
 
Figure 1: Prepare Step Assessment Configuration Tab
 
 
 
 
Under the Features heading scroll to Standard and select it.
 
 
 
 
 
 
 
 
From the Cybersecurity Standards Selection list, scroll down until you find American Water Works Association, and select. You can use the drop-down arrows at the right to expand the headings for more information.
 
 
Figure 2: Prepare Step Cybersecurity Standards Selection Tab
 
 
 
 
 
 
 
 
 
 
AWWA Cybersecurity Tool Dialogue Box Pop-Up
 
 
After making your selection, the AWWA Cybersecurity Tool dialogue pop-up appears:
 
 
Figure 3: AWWA Tool Dialogue Pop-Up
 
 
 
 
 
 
 
 
 
 
 
 
 
 
Select "Yes" to open your file explorer and upload your AWWA Cybersecurity Tool Excel file. Uploading your sheet will populate the CSET Assessment tab with the questions and answers you previously provided in the AWWA Tool.
 
 
Selecting "No" will redirect AWWA site so that you can complete the questions via the Cyber Tool.
 
Select "Continue Anyway" to load the AWWA standard in CSET to directly answer assessment questions in the Assessment tab. This option requires you to directly answer each assessment question without the benefit of your AWWA Tool data import.
 
 
 
Assessment Tab
 
In the Standard Questions screen, choose Questions Mode at the top of the screen. This allows you to input your assessment responses directly to the CSET tool:
 
 
NOTE: The AWWA standard is best utilized in “Questions” mode as opposed to “Requirements” mode.
